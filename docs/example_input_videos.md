# List of example videos

Feel free to use these videos when starting out with Visimusic!

All videos available for download from [PixaBay](https://pixabay.com/).

It is recommended to download the videos at lower resolutions for optimal performance.
There is no benefit in high-resolution video as Visimusic will still render it in a small view finder.
