# Dev Journal Entry 12

### Brief work description

A lot of work done over the past two weeks, and it really feels like the project is finally coalescing into something great.
I went from a basically useless demo to a feature-rich plugin in a very short time, all thanks to the architecture design.

I am pretty happy with the scale of visual features we are covering and also the effects. But some of them have room for improvement
That is in my opinion the final thing to focus on. Adding even more features is not a priority anymore.

Yesterday I added a base frequency slider, thinking it would be a nice thing to have control over, but it completely blew 
me away with the sound that it creates when it is moved. It might be worth to map it to some visual feature (or at least give the option).
